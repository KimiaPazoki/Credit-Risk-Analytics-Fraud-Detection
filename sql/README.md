# SQL Scripts

This folder contains SQL scripts used to support the Credit Risk Analytics & Fraud Detection project.

## Files

### credit-risk-kpis.sql
Calculates key portfolio and credit risk metrics, including total applications, approval rate, loan exposure, average credit score, and average risk probability.

### fraud-detection-analysis.sql
Analyzes fraud activity by region, including total transactions, fraud transactions, fraud rate, and fraud exposure.

### customer-segmentation.sql
Segments customers by credit score band and evaluates default history, average credit score, and debt-to-income ratio.

## Purpose
These SQL scripts demonstrate data extraction, KPI calculation, aggregation, segmentation, and analytical querying for risk and fraud reporting.
